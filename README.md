# Kunitz Domain HMM Pipeline and Structural Bioinformatics Workflow
This repository contains a reproducible bioinformatics workflow for the construction, optimization, and validation of a Hidden Markov Model (HMM) dedicated to the identification of Kunitz-type protein domains. The pipeline integrates structural bioinformatics, sequence analysis, Linux command-line utilities, Python scripting, and HMMER tools to provide an automated framework for protein domain classification and benchmarking.

The workflow includes structural alignment with mTM-align, profile HMM construction with HMMER, dataset curation, parsing of search outputs, and performance evaluation through cross-validation and threshold optimization.
All scripts, commands, and datasets required to reproduce the analysis are available in this repository.
